## 先搞懂：CN2到底是什么

CN2，全称 ChinaTelecom Next Carrier Network（中国电信下一代承载网），是中国电信专门为国际流量优化的高端骨干网络。

普通用户访问国际网站走的是"163骨干网"——节点多、用的人多、高峰期经常堵。CN2就是一条专门给VIP用的快车道，路由更短，节点更少，拥堵的概率低得多。

CN2内部还分两档：

**CN2 GT（Global Transit）**：去程或回程会出现202.97开头的普通节点，晚高峰偶尔也会堵，算是中端。

**CN2 GIA（Global Internet Access）**：全程走59.43开头的节点，是目前CN2中最高端的产品，路线表现最稳定，很少拥堵，但价格也最高。

对于有国内访问需求的用户，线路优先级基本上是：CN2 GIA > CN2 GT > 163直连。
